# Demo Video Script

## 2 to 5 minute walkthrough

1. Open the live app and show the landing/login flow.
2. Log in with the seeded admin account and mention role-based access.
3. Open the dashboard and point out project count, task count, completion rate, and overdue work.
4. Open a project board and create a new task.
5. Edit an existing task, change its status, then delete a test task.
6. Add a member to the project using their email.
7. Log out and show the member account to demonstrate reduced permissions.
8. Close by mentioning Railway deployment and the PostgreSQL-backed data store.

## Short narration line

"Ethara is a full-stack team task manager with authentication, role-based access, project and task management, live dashboard metrics, and Railway-ready PostgreSQL storage."
